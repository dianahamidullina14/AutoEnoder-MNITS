# AutoEnoder-MNITS
# Autoencoder for MNIST: Feature Extraction & Classification

## 📌 О проекте
Проект демонстрирует использование сверточного автоэнкодера для автоматического извлечения признаков из изображений MNIST с последующей классификацией цифр на основе полученных эмбеддингов.

**Ключевая идея**: обучить модель сжимать изображения в компактное латентное пространство без использования меток, затем применить эти признаки для задачи классификации.

---

## 🏗 Архитектура

### Автоэнкодер
| Компонент | Описание |
|-----------|----------|
| **Encoder** | 3 сверточных блока (Conv2d + BatchNorm + ReLU) с stride=2 |
| **Bottleneck** | Полносвязный слой, сжатие до 128 признаков |
| **Decoder** | 3 блока с upsample + свертки, восстановление изображения |
| **Выход** | Сигмоидная активация (пиксели 0-1) |

**Параметры**:
- Вход: 1 канал (grayscale)
- Базовых каналов: 32
- Latent dim: 128
- Всего параметров: ~230K

---

## 📊 Даные

| Выборка | Размер | Назначение |
|---------|--------|------------|
| Train | 1000 изображений | Обучение автоэнкодера |
| Validation | 10000 изображений | Оценка реконструкции + классификация |

---

## 🔬 Эксперимент

### Этапы
1. **Обучение автоэнкодера** (15 эпох, MSE loss, Adam lr=1e-3)
2. **Извлечение эмбеддингов** (128-мерные векторы)
3. **Классификация** Random Forest на эмбеддингах
